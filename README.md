# OTCi x Claude Code Masterclass

Build software with AI — no coding experience needed.

Today you'll use Claude Code, an AI assistant that writes code for you based on what you describe in plain English.

## Before We Start

Your instructor should have helped you with:
- A GitHub account (you're looking at GitHub right now!)
- An invitation to our GitHub organisation
- An invitation to the Anthropic Console (this powers Claude Code)

If you're missing any of these, raise your hand.

## Getting Started

### 1. Create your own copy of this repo

Click the green **"Use this template"** button at the top of this page, then:
- **Owner**: select `haskytech-masterclass`
- **Repository name**: `yourname-masterclass` (e.g., `alice-masterclass`)
- **Visibility**: Private
- Click **"Create repository"**

### 2. Launch your Codespace

On your new repo page:
- Click the green **"Code"** button
- Select the **"Codespaces"** tab
- Click **"Create codespace on main"**
- Wait about a minute for everything to set up

### 3. Log in to Claude Code

Once the editor loads in your browser:

**In the terminal at the bottom:**
```bash
claude
```
- Select **option 2** — "Anthropic Console account"
- A browser tab will open — sign in with your Anthropic Console account
- After signing in, the browser will show an error page — **that's expected!**
- **Copy the URL** from your browser address bar (starts with `http://localhost...`)

**Open a second terminal** (click the `+` icon in the terminal panel):
```bash
bash callback.sh
```
- Paste the URL you copied
- Press **Enter**, then **Ctrl+D**
- You should see "Login successful!"

Go back to your first terminal — Claude Code is ready.

### 4. Start building!

Not sure what to do? Just type:
```
what do I do next?
```

Claude knows the session plan and will guide you.

## Tips

- **Describe what you want** in everyday language — Claude writes the code
- **Look at the result**, then tell Claude what to change — that's the whole workflow
- **There's no wrong answer** — experiment, try things, have fun
- **To preview your website** — right-click your HTML file in the left panel → "Open with Live Server"
